GitPagesCNAME

2020-03-19
Hexo

使用 CNAME 文件解决 Hexo 发布后 GitHub Pages 自定义域名失效

本篇内容参考了博客文章hexo发布之后gitpage自定义域名失效-许盛，首先进行感谢。

使用Hexo将博客托管到GitHub Pages，每次使用 hexo deploy 发布后，使用原域名访问博客出现404

检查Settings-Github Pages下的Custom domain，发现原自定义域名被清空，重新填入后恢复访问

为了解决这种情况，可在source文件夹下新建CNAME文件，文件内填入自定义域名，重新发布

注意：CNAME文件无后缀，文件内容仅填入域名

Hexo: 4.2.0
OS: Win10 1909
