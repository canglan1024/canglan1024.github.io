1.符号转义

Markdown语法通过特殊符号实现，当需要单独显示这些符号时，可在特殊符号前增加转义符号\，同时通过该符号所设置的格式也不会被解析。

加粗
**转义**

Markdown中需要使用转义显示的符号如下：

Fuhao type
\ 反斜线
` 反引号
* 星号
_ 下划线
() 小括号
[] 方括号
{} 花括号
# 井号
+ 加号
- 减号
. 小数点
! 感叹号

定义模块

其中tag为模块名，type为模块类型，共有6种：

Type 说明
start 开始
end 结束
operation 操作
subroutine 子程序
condition 条件
inputoutput 输入或输出

content指框图种显示的文本，注意type:与文本之间须有一个空格。
url是与文本绑定的链接。
